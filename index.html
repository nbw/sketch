Sketch OPZ

Etch A Sketch meets the OP-Z.

REQUIRES AN OP-Z SYNTHESIZER.

Instructions

  1. Plug in your OP-Z into your computer
  2. Make sure it's set to output midi (I used the iOS app to do this)
  3. Click on the connect button

OP-Z Controls

A number of the params are controllable with the OP-Z directly. Specifically using the MOTION track (far right).

  • GREEN DIAL: ← / → ・left/right
  • BLUE DIAL: brush size
  • YELLOW DIAL: erase (fully clockwise)
  • RED DIAL: ↑ / ↓ ・ up/down

Just try and you'll get it.

Browser Support

Unfortunately Web MIDI is supported by a limited number of browsers.

tl;dr, I'd recommend using Chrome.
